Sobre a Versão Windows

Para instalar, basta baixar o instalador, descompactar e executar as instruções de instalação. O Nirooh é compatível com sistemas de 64 bits.

Baixe o instalador

Ou se preferirm basta baixar o pacote correspondente à sua distribuição e seguir as instruções de instalação do CMS. O Nirooh é compatível com sistemas de 64 bits.
