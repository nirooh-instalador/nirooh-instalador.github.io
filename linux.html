Nirooh para Linux

Sobre a Versão Linux

O Nirooh Watchdog para Linux oferece todas as funcionalidades da versão Windows, adaptadas para funcionar perfeitamente em sistemas operacionais baseados em Linux. Nossa versão Linux é otimizada para desempenho e compatibilidade com as principais distribuições do mercado.

Requisitos do Sistema

  • Sistema operacional Linux 64-bit
  • 2GB de RAM (mínimo)
  • 100MB de espaço em disco
  • Conectividade com a internet
